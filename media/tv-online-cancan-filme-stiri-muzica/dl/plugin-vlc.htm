Instalarea pluginului VLC in Windows
 
[1] : descarcati ultima versiune VLC
 
  • Windows 2000 (SP4 + UR1), XP, Vista, 7, 8 :
    - 32 biti : VLC Windows x32
    - 64 biti : VLC Windows x64
 
  • Windows 95, 98, ME :
    - Puteti sa instalati VLC 2.1.5 numai dupa ce instalati KernelEx.
    - Daca nu doriti sa instalati KernelEx, folositi una dintre variantele mai vechi de VLC: versiuni VLC / download (Recomandam versiunea 0.8.0.)
 
[2] : Dupa desccarcare, rulati installerul (daca nu se lanseaza automat) din folderul in care l-ati descarcat.
  Atentie : nu uitati sa bifati mozilla / firefox plugin si ActiveX plugin :
 
 
[3] : Dupa ce terminati instalarea, inchideti programul de Setup si toate ferestrele de browser pentru a activa pluginul.
 
Instalarea pluginului VLC pe MAC
 
[1] : Va rugam sa inchideti browserul complet inainte de instalare!
 
[2] : Pluginul corespunzator se va selecta in functie de versiunea de Mac OS si CPU.
 
 
  • Mac OS X versiunea 10.5 si mai noi :
    - Procesor Mac Intel : macosx
    - Procesor Mac PowerPC : mac PowerPC
 
[3] : Dupa download, rulati programul de instalare (in Safari porneste automat) selectand mai intai
  vlc-plugin-nnnn.dmg (nnnn depinde de versiunea de plugin), apoi fisierul VLC Internet Plug-In.pkg continut in imaginea de disc.
 
[4] : Dupa instalare, este necesar un restart al Mac-ului, pentru a activa pluginul.
